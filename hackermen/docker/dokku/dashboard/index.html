üöÄ Getting Started

  1. SSH into your server: ssh root@dokku.brainiac.gg
  2. Create an app: dokku apps:create myapp
  3. On your local machine, add the remote: git remote add dokku dokku@dokku.brainiac.gg:myapp
  4. Deploy: git push dokku main
  5. Your app will be available at: http://myapp.dokku.brainiac.gg
